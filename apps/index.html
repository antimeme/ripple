Ripple Apps

Ripple contains a collection of experimental web apps.

Asteroids
A JavaScript clone of the classic arcade space game. (And a WebAssembly version.)
Streya
Prototype of a tile based space ship game
Engine Calculator
Computes delta-v using the Tsiolkovsky rocket equation. Hey, it's not brain surgery.
Multi-vector Demonstration
Shows off intersection calculations using Geometric Algebra
Quanta
A simplistic representation of quantum mechanics
On the Ball
Sphere approximation by subdividing regular polyhedra
Eight Queens
Solve the eight queens problem
Footy
A tool for creating diagrams of plays in international football (sometimes called soccer in less civilized countries)
Whiplash Paradox
Prototype of a game about time travel
Ball
A simple animation of a bouncing ball -- with sound!
Tic-Tac-Toe
A simple implementation of a simple game
Jarbles
Java applet implementation of Abalone (obsolete)
Grid Test
Interactive demonstraton of the grid library
Gestur
A tool for exploring the Gestur library
Poke!
A tool for exploring mouse and touch events
