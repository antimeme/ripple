Nothing But Set

Basic Set Theory

  • Empty: {} ≝ ∀a : a ∉ {}
  • Union: S∪T ≝ { ∀a : a∈S ∨ a∈T }
    Example: {a, b}∪{b, c} = {a, b, c}
  • Intersection: S∩T ≝ { ∀a : a∈S ∧ a∈T }
    Example: {a, b}∩{b, c} = {b}
  • Subset: S⊆T ≝ ∀a∈S : a ∈ T
    Example: {a, b}⊆{a, b, c}⊆{a, b, c}
  • Equality: S=T ≝ (S⊆T ∧ T⊆S)
    • Reflexive: ∀S : S = S
    • Symmetric: ∀S,T : (S = T) ⟺ (T = S)
    • Transitive: ∀S,T,U : (S=U ∧ T=U) ⟹ (S = T)
    • Unordered: ∀a,b : {a, b} = {b, a}
    • Repetition: ∀a : {a, a} = {a}

Natural Numbers (ℕ)

  • succ(a) ≝ a ∪ {a}
  • ℕ ≝ { ∀a : a = {} ∨ (∃b∈ℕ : a = succ(b)) }

    • 2 ≝ succ(1) = {0, 1} = {{}, {{}}}
    • 3 ≝ succ(2) = {0, 1, 2} = {{}, {{}}, {{}, {{}}}}

Cardinality

  • Given f : X ⟶ Y
    • Injection: ∀a,b∈X : f(a) = f(b) ⟹ a = b
    • Surjection: ∀a∈Y ∃b∈X : b = f(a)
    • Bijection: injection and surjection
  • Two sets with a bijection have the same cardinality
  • Example: sets with 3 elements have a bijection with 3ℕ

Addition ℕ

  • Definition:
    • Base: a + 0 ≝ a
    • Induction: a + succ(b) ≝ succ(a + b)
  • Example: 3 + 2
    • 3 + succ(1) = 3 + succ(succ(0)) = succ(succ(3 + 0))
    • succ(succ(3)) = succ(4) = 5
  • Closure: ∀a,b∈ℕ : a + b ∈ℕ
  • Associative: ∀a,b,c∈ℕ : (a + b) + c = a + (b + c)
  • Commutative: ∀a,b∈ℕ : a + b = b + a
  • Identity: ∀a∈ℕ a + 0 = a

Multiplication ℕ

  • Definition:
    • a × 0 ≝ 0
    • a × succ(b) ≝ a + (a × b)
  • Example: 3 × 2
    • 3 × succ(1) = 3 × succ(succ(0))
    • 3 + 3 + (3 × 0) = 3 + 3 = 6
  • Closure: ∀a,b∈ℕ : a × b ∈ℕ
  • Associative: ∀a,b,c∈ℕ : (a × b) × c = a × (b × c)
  • Commutative: ∀a,b∈ℕ : a × b = b × a
  • Identity: ∀a∈ℕ a × 1 = a
  • Distributive: ∀a,b,c∈ℕ :
    (a × (b + c) = (a × b) + (a × c)) ∧
    ((b + c) × a = (b × a) + (c × a))

Ordered Pairs

A set has no meaningful order, but order can be useful. Fortunately we can construct order using unordered sets.

  • Kuratowski 1921: (a, b) ≝ {{a}, {a, b}}
  • a is the first element of p: ∀S∈p : a∈S
  • b is the second element of p:
    (∃S∈p : b∈S) ∧ (∀S1,S2∈p : S1 ≠ S2 ⟹ b≠S1 ∧ b≠S2)
  • ∀a,b (a ≠ b) ⟹ ((a, b) ≠ (b, a))
    {{a}, {a, b}} ≠ {{b}, {a, b}}

Integers (ℤ)

  • ℤ ≝ { ∀a,b∈ℕ : (a, b) }
  • Equality: ∀a,b∈ℤ (a =ℤ b) ≝ ∃c∈ℕ : ((a1 + c, a2 + c) = (b1, b2)) ∨ ((a1, a2) = (b1 + c, b2 + c)))
  • [a - b] ≝ {∀c∈ℤ : c =ℤ (a, b)}
  • Example: -4 = (0, 4) = {{{}}, {{}, { {}, {{}}, {{}, {{}}}, {{}, {{}}, {{}, {{}}}}}}}

Addition ℤ

  • Definition: ∀ a,b∈ℤ a + b ≝ [a1 - a2] + [b1 - b2] =
    [(a1 + b1) - (a2 + b2)]
  • Closure: ∀a,b∈ℤ : a + b ∈ℤ
  • Associative: ∀a,b,c∈ℤ : (a + b) + c = a + (b + c)
  • Commutative: ∀a,b∈ℤ : a + b = b + a
  • Identity: ∀a∈ℤ a + 0 = a
  • Inverses: ∀a∈ℤ : -a ≝ [a2 - a1] : a = [a1 - a2] ∧ a + (-a) = 0

Multiplication ℤ

  • Definition: ∀ a,b∈ℤ a × b ≝ [a1 − a2] × [b1 − b2] =
    [(a1 × b1 + a2 × b2) − (a2 × b1 + a1 × b2)]
  • Closure: ∀a,b∈ℤ : a × b ∈ℤ
  • Associative: ∀a,b,c∈ℤ : (a × b) × c = a × (b × c)
  • Commutative: ∀a,b∈ℤ : a × b = b × a
  • Distributive: ∀a,b,c∈ℤ :
    (a × (b + c) = (a × b) + (a × c)) ∧
    ((b + c) × a = (b × a) + (c × a))
  • Identity: ∀a∈ℤ a × 1 = a

Groups (G, ∘)

A group is a set G and a binary operation G ∘ G ⟹ G with four properties:

  • Closure: ∀a,b∈G : a ∘ b ∈G
  • Associative: (a ∘ b) ∘ c = a ∘ (b ∘ c)
  • Identity: ∃e∈G : ∀a∈G a ∘ e = a
  • Inverses: ∀a∈G ∃a-1∈G : a ∘ a-1 = e

The definition of a group requires that the identity and inverses work when applied from the right. Without knowing anything else about the group we can prove that these values work from the left as well.

  • e = a-1 ∘ (a-1)-1 (Inverses)
  • e = (a-1 ∘ e) ∘ (a-1)-1 (Identity)
  • e = (a-1 ∘ (a ∘ a-1)) ∘ (a-1)-1 (Inverses)
  • e = ((a-1 ∘ a) ∘ a-1) ∘ (a-1)-1 (Associativity)
  • e = (a-1 ∘ a) ∘ (a-1 ∘ (a-1)-1) (Associativity)
  • e = (a-1 ∘ a) ∘ e (Identity)

So inverses work regardless of which side they're applied. This means the identity also works from both sides:

  • e ∘ a = (a ∘ a-1) ∘ a (Inverses)
  • e ∘ a = a ∘ (a-1 ∘ a) (Associativity)
  • e ∘ a = a ∘ e (Inverses)
  • e ∘ a = a (Identity)

A group is the simplest mathematical structure that allows algebra. Suppose we have know a, b and x ∘ a = b:

  • (x ∘ a) ∘ a-1 = b ∘ a-1 (Transitive Equality)
  • x ∘ (a ∘ a-1) = b ∘ a-1 (Associativity)
  • x ∘ e = b ∘ a-1 (Inverses)
  • x = b ∘ a-1 (Identity)

Abelian Groups

An abelian group is a group with a commutative operation. Integers with addition form an abelian group: (ℤ, +).

Rings (R, ∘, ⊚)

A ring is a set and two binary operations R ∘ R ⟹ R and R ⊚ R ⟹ R with four properties:

  • (R, ∘) is an abelian group
  • Closure: ∀a,b∈R : a ⊚ b ∈ R
  • Associative: (a ⊚ b) ⊚c = a ⊚ (b ⊚ c)
  • Distributive: ∀a,b,c∈R :
    (a ⊚ (b ∘ c) = (a ⊚ b) ∘ (a ⊚ c)) ∧
    ((b ∘ c) ⊚ a = (b ⊚ a) ∘ (c ⊚ a))

Commutative Rings

A commutative ring is a with a commutative operations. Integers with addition and multiplication form a commutative ring (ℤ, +, ×).

Unitary Rings (U, ∘, ⊚)

A unitary ring is a ring with an identity element for the second operation:

  • (U, ∘, ⊚) is a ring
  • Identity: ∃e∈U ∀a∈U : a ⊚ e = a

Integers with addition and multiplication form a unitary ring (ℤ, +, ×).

Rational Numbers (ℚ)

  • ℚ ≝ { ∀a∈ℤ,b≠0∈ℤ : (a, b) }
  • Equality: ((a1, a2) =ℚ (b1, b2)) ≝ ∃c≠0∈ℤ : ((a1 × c, a2 × c) = (b1, b2)) ∨ ((a1, a2) = (b1 × c, b2 × c)))
  • [a / b] ≝ {∀c∈ℚ : c =ℚ (a, b)}
  • Example: -4/3 = (-4, 3) = {{-4}, {-4, 3}} = {{{{{}}, {{}, {{}, {{}}, {{}, {{}}}, {{}, {{}}, {{}, {{}}}}}}}}, {{{{}}, {{}, {{}, {{}}, {{}, {{}}}, {{}, {{}}, {{}, {{}}}}}}}, {{}, {{}}, {{}, {{}}}}}}

Addition ℚ

  • Definition: ∀a,b∈ℚ a + b ≝ (a1, a2) + (b1, b2) =
    ((a1 × b2 + b1 × a2), (a2 × b2))
  • Closure: ∀a,b∈ℚ : a + b ∈ℚ
  • Associative: ∀a,b,c∈ℚ : (a + b) + c = a + (b + c)
  • Commutative: ∀a,b∈ℚ : a + b = b + a
  • Identity: ∀a∈ℚ a + 0 = a
  • Inverses: ∀a∈ℚ : -a ≝ (-a1, a2) : a = (a1, a2) ∧ a + (-a) = 0

Multiplication ℚ

  • Definition: ∀a,b∈ℚ a × b ≝ (a1, a2) × (b1, b2) = (a1 × b1, a2 × b2)
  • Closure: ∀a,b∈ℚ : a × b ∈ℚ
  • Associative: ∀a,b,c∈ℚ : (a × b) × c = a × (b × c)
  • Commutative: ∀a,b∈ℚ : a × b = b × a
  • Distributive: ∀a,b,c∈ℚ :
    (a × (b + c) = (a × b) + (a × c)) ∧
    ((b + c) × a = (b × a) + (c × a))
  • Identity: ∀a∈ℚ a × 1 = a
  • Inverses: ∀a∉[0 / 1]∈ℚ : a-1 ≝ (a2, a1) : a = (a1, a2) ∧ a × a-1 = 1

Fields (F, ∘, ⊚)

A field is a unitary ring with inverses:

  • (F, ∘, ⊚) is a unitary ring
  • Inverses: ∀a∈U a ≠ e∘ ⟹ ∃a-1∈U : a ⊚ a-1 = e⊚

An example of a field is rational numbers with addition and multiplication: (ℚ, +, ×)
