Naive Set Theory

A set is as simple structure defined by the members it contains. Let's see what we can construct with informal definitions.

  • Empty: {} ≝ ∀a : a ∉ {}
  • Union: S∪T ≝ { ∀a : a∈S ∨ a∈T }
    Example: {a, b}∪{b, c} = {a, b, c}
  • Intersection: S∩T ≝ { ∀a : a∈S ∧ a∈T }
    Example: {a, b}∩{b, c} = {b}
  • Subset: S⊆T ≝ ∀a∈S : a ∈ T
    Example: {a, b}⊆{a, b, c}⊆{a, b, c}
  • Equality: S=T ≝ (S⊆T ∧ T⊆S)
    • Reflexive: ∀S : S = S
    • Symmetric: ∀S,T : (S = T) ⟺ (T = S)
    • Transitive: ∀S,T,U : (S=U ∧ T=U) ⟹ (S = T)
  • Repetition: ∀a : {a, a} = {a}

Ordered Pairs

Sets have no order. ∀a,b : {a, b} = {b, a} Creating order with sets requires some creativity.

  • Kuratowski 1921: (a, b) ≝ {{a}, {a, b}}
  • a is the first element of p: ∀S∈p : a∈S
  • b is the second element of p:
    (∃S∈p : b∈S) ∧ (∀S1,S2∈p : S1 ≠ S2 ⟹ b≠S1 ∧ b≠S2)
  • ∀a,b (a ≠ b) ⟹ ((a, b) ≠ (b, a))
    {{a}, {a, b}} ≠ {{b}, {a, b}}

Cardinality

  • Relation: (r : D ⟶ R) ≝ { (a, b) : a∈D,b∈R }
  • Application: r(a) = b ⟺ (a, b)∈r
  • Given relation r : D ⟶ R
    • Injection: ∀a,b∈D : r(a) = r(b) ⟹ a = b
    • Surjection: ∀a∈R ∃b∈D : r(b) = a
    • Bijection: injection and surjection
  • Two sets with a bijection have the same cardinality
  • Example: sets with 3 elements have a bijection with 3ℕ

Addition ℕ

  • Definition:
    • Base: ∀a∈ℕ a + 0 ≝ a
    • Step: ∀a,b∈ℕ a + succ(b) ≝ succ(a + b)
  • Example: 3 + 2
    • 3 + 2 = 3 + succ(1) :: Definition of 2
    • 3 + 2 = 3 + succ(succ(0)) :: Definition of 1
    • 3 + 2 = succ(3 + succ(0)) :: Definition Step
    • 3 + 2 = succ(succ(3 + 0)) :: Definition Step
    • 3 + 2 = succ(succ(3)) :: Definition Base
    • 3 + 2 = succ(4) :: Successor
    • 3 + 2 = 5 :: Successor
  • Closed: ∀a,b∈ℕ : a + b ∈ℕ
  • Associative: ∀a,b,c∈ℕ : (a + b) + c = a + (b + c)
    • (a + b) + 0 = a + b = a + (b + 0) (Definition Base)
    • (a + b) + c = a + (b + c) (Assumption)
    • (a + b) + succ(c) = succ((a + b) + c) (Definition Step)
    • (a + b) + succ(c) = succ(a + (b + c)) (Induction)
    • (a + b) + succ(c) = a + succ(b + c) (Definition Step)
    • (a + b) + succ(c) = a + (b + succ(c)) (Definition Step)
  • Identity: ∀a∈ℕ a + 0 = a = 0 + a
    • 0 + 0 = 0 :: Definition Base
    • 0 + a = a :: Assumption
    • 0 + succ(a) = succ(0 + a) :: Definition Step
    • 0 + succ(a) = succ(a) :: Induction
  • Commutative: ∀a,b∈ℕ : a + b = b + a
    • Successor + 1: ∀a∈ℕ succ(a) = a + 1
      • succ(a) = succ(a + 0) (Definition Base)
      • succ(a) = a + succ(0) (Definition Step)
      • succ(a) = a + 1 (Definition of 1)
    • Commutativity of 1: ∀a∈ℕ a + 1 = 1 + a
      • 0 + 1 = 1 + 0 (Identity)
      • a + 1 = 1 + a (Assumption)
      • succ(a) + 1 = succ(a) + succ(0) (Definition of 1)
      • succ(a) + 1 = succ(succ(a) + 0) (Defintion Step)
      • succ(a) + 1 = succ((a + 1) + 0) (Successor + 1)
      • succ(a) + 1 = succ(a + 1) (Defintion Base)
      • succ(a) + 1 = succ(1 + a) (Induction)
      • succ(a) + 1 = 1 + succ(a) (Definition Step)
    • a + b = b + a (Assumption)
    • a + succ(b) = a + (b + 1) (Successor + 1)
    • a + succ(b) = (a + b) + 1 (Associativity)
    • a + succ(b) = (b + a) + 1 (Induction)
    • a + succ(b) = b + (a + 1) (Associativity)
    • a + succ(b) = b + (1 + a) (Commutativity of 1)
    • a + succ(b) = (b + 1) + a (Associativity)
    • a + succ(b) = succ(b) + a (Successor + 1)

Chapter 2: Abstract Algebra

Magma (M, ∘)

A magma is a set M and a binary operation M ∘ M ⟹ M with one important property:

  • Closed: ∀a,b∈M : a ∘ b ∈M

Groups (G, ∘)

A group is a set G and a binary operation G ∘ G ⟹ G with four properties:

  • Closed: ∀a,b∈G : a ∘ b ∈G
  • Associative: (a ∘ b) ∘ c = a ∘ (b ∘ c)
  • Identity: ∃e∈G : ∀a∈G a ∘ e = a
  • Inverses: ∀a∈G ∃a-1∈G : a ∘ a-1 = e

The definition of a group requires that the identity and inverses work when applied from the right. Without knowing anything else about the group we can prove that these values work from the left as well.

  • e = a-1 ∘ (a-1)-1 (Inverses)
  • e = (a-1 ∘ e) ∘ (a-1)-1 (Identity)
  • e = (a-1 ∘ (a ∘ a-1)) ∘ (a-1)-1 (Inverses)
  • e = ((a-1 ∘ a) ∘ a-1) ∘ (a-1)-1 (Associativity)
  • e = (a-1 ∘ a) ∘ (a-1 ∘ (a-1)-1) (Associativity)
  • e = (a-1 ∘ a) ∘ e (Identity)

So inverses work regardless of which side they're applied. This means the identity also works from both sides:

  • e ∘ a = (a ∘ a-1) ∘ a (Inverses)
  • e ∘ a = a ∘ (a-1 ∘ a) (Associativity)
  • e ∘ a = a ∘ e (Inverses)
  • e ∘ a = a (Identity)

A group is the simplest mathematical structure that allows algebra. Suppose we have know a, b and x ∘ a = b:

  • (x ∘ a) ∘ a-1 = b ∘ a-1 (Transitive Equality)
  • x ∘ (a ∘ a-1) = b ∘ a-1 (Associativity)
  • x ∘ e = b ∘ a-1 (Inverses)
  • x = b ∘ a-1 (Identity)

Abelian Groups

An abelian group is a group with a commutative operation. Integers with addition form an abelian group: (ℤ, +).

Rings (R, ∘, ⊚)

A ring is a set and two binary operations R ∘ R ⟹ R and R ⊚ R ⟹ R with four properties:

  • (R, ∘) is an abelian group
  • Closed: ∀a,b∈R : a ⊚ b ∈ R
  • Associative: (a ⊚ b) ⊚c = a ⊚ (b ⊚ c)
  • Distributive: ∀a,b,c∈R :
    (a ⊚ (b ∘ c) = (a ⊚ b) ∘ (a ⊚ c)) ∧
    ((b ∘ c) ⊚ a = (b ⊚ a) ∘ (c ⊚ a))

Commutative Rings

A commutative ring is a with a commutative operations. Integers with addition and multiplication form a commutative ring (ℤ, +, ×).

Unitary Rings (U, ∘, ⊚)

A unitary ring is a ring with an identity element for the second operation:

  • (U, ∘, ⊚) is a ring
  • Identity: ∃e∈U ∀a∈U : a ⊚ e = a

Integers with addition and multiplication form a unitary ring (ℤ, +, ×).

Fields (F, ∘, ⊚)

A field is a unitary ring with inverses:

  • (F, ∘, ⊚) is a unitary ring
  • Inverses: ∀a∈U a ≠ e∘ ⟹ ∃a-1∈U : a ⊚ a-1 = e⊚

An example of a field is rational numbers with addition and multiplication: (ℚ, +, ×)
