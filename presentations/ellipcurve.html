Groups ⟬G, ∘⟭

A group is a set G and a binary operation G ∘ G ⟶ G with four properties:

  • Closed: ∀a,b∈G : a ∘ b ∈G
  • Associative: ∀a,b,c∈G (a ∘ b) ∘ c = a ∘ (b ∘ c)
  • Identity: ∃e∈G ∀a∈G a ∘ e = a
  • Inverses: ∀a∈G ∃a-1∈G a ∘ a-1 = e

By definition identity and inverses work when applied from the right. Group axioms are enough to prove that these also work from the left.

  • e = a-1 ∘ (a-1)-1 :: Inverses
  • e = (a-1 ∘ e) ∘ (a-1)-1 :: Identity
  • e = (a-1 ∘ (a ∘ a-1)) ∘ (a-1)-1 :: Inverses
  • e = ((a-1 ∘ a) ∘ a-1) ∘ (a-1)-1 :: Associativity
  • e = (a-1 ∘ a) ∘ (a-1 ∘ (a-1)-1) :: Associativity
  • e = (a-1 ∘ a) ∘ e :: Inverses
  • e = a-1 ∘ a :: Identity

So inverses work regardless of which side they're applied. This means the identity also works from both sides:

  • e ∘ a = (a ∘ a-1) ∘ a :: Inverses
  • e ∘ a = a ∘ (a-1 ∘ a) :: Associativity
  • e ∘ a = a ∘ e :: Inverses
  • e ∘ a = a :: Identity

A group is the simplest mathematical structure that allows algebra. Suppose we have know a, b and
(x ∘ a = b):

  • (x ∘ a) ∘ a-1 = b ∘ a-1 :: Equality
  • x ∘ (a ∘ a-1) = b ∘ a-1 :: Associativity
  • x ∘ e = b ∘ a-1 :: Inverses
  • x = b ∘ a-1 :: Identity

Abelian Group ⟬G, ∘⟭

An abelian group is a group with a commutative operation.

  • ⟬G, ∘⟭ is a group
  • Commutative: ∀a,b,c∈G (a ∘ b) = (b ∘ a)

Integers with addition form an abelian group: ⟬ℤ, +⟭

Rings ⟬R, ∘, ⊚⟭

A ring is a set and two binary operations R ∘ R ⟶ R and R ⊚ R ⟶ R with four properties:

  • ⟬R, ∘⟭ is an abelian group
  • Closed: ∀a,b∈R : a ⊚ b ∈ R
  • Associative: (a ⊚ b) ⊚c = a ⊚ (b ⊚ c)
  • Distributive: ∀a,b,c∈R
    (a ⊚ (b ∘ c) = (a ⊚ b) ∘ (a ⊚ c)) ∧
    ((b ∘ c) ⊚ a = (b ⊚ a) ∘ (c ⊚ a))

Commutative Rings

A commutative ring is a with commutative operations.

  • ⟬R, ∘, ⊚⟭ is a ring
  • Commutative: ∀a,b,c∈R (a ⊚ b) = (b ⊚ a)

Integers with addition and multiplication form a commutative ring (ℤ, +, ×)

Unitary Rings ⟬U, ∘, ⊚⟭

A unitary ring is a ring with an identity element for the second operation:

  • ⟬U, ∘, ⊚⟭ is a ring
  • Identity: ∃e⊚∈U ∀a∈U a ⊚ e⊚ = a

Integers with addition and multiplication form a unitary ring ⟬ℤ, +, ×⟭

Fields ⟬F, ∘, ⊚⟭

A field is a commutative, unitary ring with inverses:

  • ⟬F, ∘, ⊚⟭ is a unitary, commutative ring
  • Inverses: ∀a∈U a ≠ e∘ ⟹ ∃a-1∈U a ⊚ a-1 = e⊚

An example of a field is rational numbers with addition and multiplication: ⟬ℚ, +, ×⟭
