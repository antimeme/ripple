Group ⟬G, ∘⟭

A group is a set G and a binary operation G ∘ G ⟶ G with four properties:

  • Closed: ∀a,b∈G : a ∘ b ∈G
  • Associative: ∀a,b,c∈G (a ∘ b) ∘ c = a ∘ (b ∘ c)
  • Identity: ∃e∈G ∀a∈G a ∘ e = a
  • Inverses: ∀a∈G ∃a-1∈G a ∘ a-1 = e

Abstraction

By definition identity and inverses work when applied from the right. Group axioms are enough to prove that these also work from the left.

  • Identity: ∃e∈G ∀a∈G a ∘ e = a
  • Inverses: ∀a∈G ∃a-1∈G a ∘ a-1 = e

Proof: Commutative Inverses

  • e = a-1 ∘ (a-1)-1 :: Inverses
  • e = (a-1 ∘ e) ∘ (a-1)-1 :: Identity
  • e = (a-1 ∘ (a ∘ a-1)) ∘ (a-1)-1 :: Inverses
  • e = ((a-1 ∘ a) ∘ a-1) ∘ (a-1)-1 :: Associativity
  • e = (a-1 ∘ a) ∘ (a-1 ∘ (a-1)-1) :: Associativity
  • e = (a-1 ∘ a) ∘ e :: Inverses
  • e = a-1 ∘ a :: Identity

Proof: Commutative Identity

  • e ∘ a = (a ∘ a-1) ∘ a :: Inverses
  • e ∘ a = a ∘ (a-1 ∘ a) :: Associativity
  • e ∘ a = a ∘ e :: Inverses
  • e ∘ a = a :: Identity

Equation Solving

A group is the simplest mathematical structure that allows algebra. Suppose we have know a, b and
(x ∘ a = b):

  • (x ∘ a) ∘ a-1 = b ∘ a-1 :: Equality
  • x ∘ (a ∘ a-1) = b ∘ a-1 :: Associativity
  • x ∘ e = b ∘ a-1 :: Inverses
  • x = b ∘ a-1 :: Identity

Abelian Group ⟬G, ∘⟭

An abelian group is a group with a commutative operation.

  • ⟬G, ∘⟭ is a group
  • Commutative: ∀a,b,c∈G (a ∘ b) = (b ∘ a)

Integers with addition form an abelian group:
⟬ℤ, +⟭

Ring ⟬R, ∘, ⊚⟭

A ring is a set and two binary operations R ∘ R ⟶ R and R ⊚ R ⟶ R with four properties:

  • ⟬R, ∘⟭ is an abelian group
  • Closed: ∀a,b∈R : a ⊚ b ∈ R
  • Associative: (a ⊚ b) ⊚c = a ⊚ (b ⊚ c)
  • Distributive: ∀a,b,c∈R
    (a ⊚ (b ∘ c) = (a ⊚ b) ∘ (a ⊚ c)) ∧
    ((b ∘ c) ⊚ a = (b ⊚ a) ∘ (c ⊚ a))

Commutative Ring ⟬R, ∘, ⊚⟭

A commutative ring is a with commutative operations.

  • ⟬R, ∘, ⊚⟭ is a ring
  • Commutative: ∀a,b,c∈R (a ⊚ b) = (b ⊚ a)

Integers with addition and multiplication form a commutative ring ⟬ℤ, +, ×⟭

Unitary Ring ⟬R, ∘, ⊚⟭

A unitary ring is a ring with an identity element for the second operation:

  • ⟬R, ∘, ⊚⟭ is a ring
  • Identity: ∃e⊚∈R ∀a∈R a ⊚ e⊚ = a

Integers with addition and multiplication form a unitary ring ⟬ℤ, +, ×⟭

Field ⟬F, ∘, ⊚⟭

A field is a commutative, unitary ring with inverses:

  • ⟬F, ∘, ⊚⟭ is a unitary, commutative ring
  • Inverses: ∀a∈U a ≠ e∘ ⟹ ∃a-1∈U a ⊚ a-1 = e⊚

An example of a field is rational numbers with addition and multiplication: ⟬ℚ, +, ×⟭

Finite Field ⟬F, ∘, ⊚⟭

A finite field is a field with a finite number of members. An example a field is integers modulo a prime integer with addition and multiplication: ⟬ℤ/pℤ, +, ×⟭

Elliptic Curve

Source: https://en.wikipedia.org/wiki/Elliptic_curve

Formula

Weierstrass Form

y2 = x3 + ax + b

Point Addition

P + Q = R

𝜆 = (yq - yp) / (xq - xp)

xr = 𝜆2 - xp - xq

yr = 𝜆(2 - xp - xr) - yp

This always results in another point on the curve.

Point Doubling

P + P = R

𝜆 = (3xpxp + a) / 2yp

xr = 𝜆2 - 2xp

yr = 𝜆(2 - xp - xr) - yp

Point addition divides by zero when adding a point to itself, so we use the tangent line.

The Point at Infinity

A conceptual point at infinity serves as the identity element.

Point Compression

Given the x coordinate and the sign of the y coordinate, the y coordinate can be computed using the curve formula.

y2 = x3 + ax + b

Finite Field Square Root

Algorithms like Tonelli-Shanks can compute square roots in modular arithetic.

Diffie-Hellman on Elliptic Curves

Integer Diffie-Hellman

Agree on parameters g and p where p is prime and g is a primitive root mod p.

Integer Diffie-Hellman

Alice chooses a and computes ga mod p

Bob chooses b and computes gb mod p

Alice gives Bob ga mod p which he raises to the power b to get gab mod p

Bob gives Alice gb mod p which she raises to the power a to get gba mod p
(which is equal to gab mod p)

Alice and Bob agree to use a modulus p = 23 and base g = 5 (which is a primitive root modulo 23).

Alice chooses a = 4, then sends Bob 54 mod 23 = 4

Bob chooses b = 3, then sends Alice 53 mod 23 = 10

Alice computes s = 104 mod 23 = 18

Bob computes s = 43 mod 23 = 18

Elliptic Curve Diffie Hellman

Agree on parameters G, p and an elliptic curve, where G is a point and p is prime (there are also binary curves which are different).

Elliptic Curve Diffie Hellman

Alice chooses a and adds G to itself a times to compute Ga mod p

Bob chooses b and adds G to itself b times to get Gb mod p

Elliptic Curve Diffie Hellman

Alice gives Bob Ga mod p to which he adds G b times to get Gab mod p

Bob gives Alice Gb mod p to which she adds G a times to get Gba mod p
(which is equal to Gab mod p)
