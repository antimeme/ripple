What is a Rotor?

A rotor is a concept from Geometric Algebra that can serve as a rotation, a complex number, a quaternion, a translation (projective geometry) and even a Lorentz boost (relativity).

But first...

Let's meet some friendly math

No seriously, there's nothing more complicated than multiplying in here.

Vectors

A vector is an oriented length (one dimension).

  • Orthonormal Basis: x, y, z
    • Ortho: orthogonal -- right angle / quarter turn
    • Normal: |x| = |y| = |z| = 1
    • Basis: v = vxx + vyy + vzz (vx, vy, vz∈ℝ)
  • ∀a∈ℝ : a v = v a = a vxx + a vyy + a vzz
  • v + u = (vx + ux)x + (vy + uy)y + (vz + uz)z

Geometric Algebra

Axiom: ∀v∈𝕍 : v v† = |v|2 ∈ℝ

We'll explain the meaning of the conjugate† operation later, but for now it's enough to know that all vectors are the same as their conjugates.

Geometric Product

Don't panic! It's just multiplication

  • v = vxx + vyy
  • u = uxx + uyy
  • v u = (vxx) (uxx) + (vxx) (uyy) + (vyx) (uxy) + (vyy) (uyy)
  • v u = vx ux x x + vx uy x y + vy ux y x + vy uy y y

Basis Collapse

Our axiom requires basis vectors to square to one.

  • x x† = x x = |x|2 = 12 = 1
  • y y† = y y = |y|2 = 12 = 1
  • z z† = z z = |z|2 = 12 = 1

Anti-Commute Basis

Consider: v v† ∈ℝ

  • v v† = vx vx x x + vx vy x y + vy vx y x + vy vy y y
  • v v† = vx vx x x + vx vy x y + vy vx y x + vy vy y y
  • v v = vx vx + vy vy + vx vy (x y + y x) ∈ℝ
  • x y + y x = 0 ⟹ x y = - y x

Geometric Product

Let's review what we've learned so far:

  • Scalars commute: ∀a∈ℝ ∀v∈𝕍 a v = v a
  • Basis Collapse: x x = y y = z z = 1
  • Anti-Commute Basis: x y = - y x

This is everything we need to compute the geometric product!

Bivectors

Bivectors have two basis vectors in each term. In three dimensions x y, y z and x z serve as a basis for bivectors in the same way that x, y and z serve as a basis for vectors. Bivectors are linear combinations of these:

∃ Bx y, By z, Bx z∈ℝ : B = Bx yx y + By zy z + Bx zx z

Trivectors

Trivectors have three basis vectors in each term. Only one basis trivectors is possible in three dimensions so they are pseudo-scalars.

∃ Tx y z∈ℝ : T = Tx y zx y z

Conjugate

The conjugate reverses the vectors in each term

  • ∀a∈ℝ a† = a
  • ∀v∈𝕍 v† = v
  • (x y)† = y x = -x y
  • (x y z)† = z y x = -z x y = x z y = -x y z
  • (x y z t)† = t z y x = -t x y z = x y z t
  • (3 + 2 x y)† = 3 - 2 x y

Geometric Product

v u = (vxx + vyy) (uxx + uyy)

  • v u = vx ux x x + vx uy x y + vy ux y x + vy uy y y
  • v u = vx ux x x + vx uy x y - vy ux x y + vy uy y y
  • v u = vx ux + vy uy + (vx uy - vy ux) x y

Geometric Product

v u = (vxx + vyy + vzz) (uxx + uyy + uzz)

  • v u = vxux x x + vxuy x y + vxuz x z + vyux y x +
    vyuy y y + vyuz y y + vzux z x + vzuy z y + vzuz z z
  • v u = vxux x x + vyuy y y + vzuz z z + vxuy x y -
    vyux x y + vyuz y z - vzuy y z + vxuz x z - vzux x z
  • v u = vxux + vyuy + vzuz + (vxuy - vyux) x y +
    (vyuz - vzuy) y z + (vxuz - vzux) x z

Dot Product

v · u ≝ (v u + u v) / 2

  • v · u = (vx ux + vy uy + (vx uy - vy ux) x y +
    ux vx + uy vy + (ux vy - uy vx) x y ) / 2 =
    (vx ux + vy uy + vx ux + vy uy ) / 2     = vx ux + vy uy
  • v · u = (vxux + vyuy + vzuz + uxvx + uyvy + uzvz) / 2 =
    (vxux + vyuy + vzuz + vxux + vyuy + vzuz ) / 2     =
    vxux + vyuy + vzuz

Wedge Product

v ^ u ≝ (v u - u v) / 2

  • v ^ u = (vx ux + vy uy + (vx uy - vy ux) x y -
    ux vx - uy vy - (ux vy - uy vx) x y) / 2 =
    (vx uy - vy ux + vx uy - vy ux ) / 2 x y     = (vx uy - vy ux) x y
  • v ^ u = (vx uy - vy ux) x y + (vy uz - vz uy) y z +
    (vx uz - vz ux) x z

Cross Product

v × u ≝ (v ^ u) z y x

  • v × u = ((vxx + vyy + vzz) ^ (uxx + uyy + uzz)) z y x
  • v × u = (vx uy - vy ux) x y z y x +
    (vy uz - vz uy) y z z y x + (vx uz - vz ux) x z z y x
  • v × u = (vy uz - vz uy)x - (vx uz - vz ux)y + (vx uy - vy ux)z

The wedge product is far more general and useful

Bivectors Norms

B = Bx yx y + By zy z + Bx zx z

B B† = Bx y2 x y y x + By z2 y z z y + Bx z2 x z z x +
(Bx y By z - By z Bx y) x y z y + (Bx y Bx z - Bx z Bx y) x y z x +
(By z Bx z - Bx z By z) y z z x = Bx y2 + By z2 + Bx z2

B B = B2 = -(Bx y2 + By z2 + Bx z2)

Unit Vectors

A unit vector has a norm of one.

û = ûxx + ûyy + ûzz

û û† = û2 = ûx2 + ûy2 + ûz2 = 1

|x|2 = |y|2 = |z|2 = 1

A unit vector describes a line through the origin

Unit Bivectors

A unit bivector has a norm of one.

U = Ux yx y + Uy zy z + Ux zx z

U U† = Ux y2 + Uy z2 + Ux z2 = 1 = -U2

|x y|2 = |y z|2 = |x z|2 = 1

A unit bivector describes a plane through the origin

Imagine That

Consider: (x y)2

  • (x y)2 = (x y) (x y)
  • (x y)2 = (x y) -(y x)
  • (x y)2 = -(x y y x) = -(x x) = -1
  • i2 = -1
  • i† = -i = -x y = y x = (x y)†

Not so imaginary after all!

In 3D

All the same properties apply to other bivectors

  • (x y)2 = (-y x)2 = -1 = i2 (i† = -i)
  • (y z)2 = (-z y)2 = -1 = j2 (j† = -j)
  • (x z)2 = (-z x)2 = -1 = k2 (k† = -k)
  • i j = (x y) (y z) = x z = k
  • j i = (y z) (x y) = -(z y x y) = (z y y x) = z x = -k

Quaternion rules suddenly seem inevitable

Inverses

v-1 = v† / (v v†)

  • v v-1 = v v† / (v v†) = |v|2 / |v|2 = 1
  • a a-1 = a a† / (a a†) = (a a) / (a a) = 1
  • (x y) (x y)-1 = (x y) (x y)† / ((x y) (x y)†) =
    (x y y x) / (x y y x)     = (x x) / (x x) = 1

    (v-1)-1 = v-1 † / (v-1 v-1 †) = v† † / (v v† v-1 v-1 †) = v

Uninvertable

Consider: M = x + 1

  • M M† = (1 + x) (1 + x)† = (1 + x) (1 + x) =
    1 + x + x + x x     = 1 + x + x + 1 = 2 + 2x Not a scalar!
  • M-1 does not exist!

Also no inverse when M M† = 0

Measure Twice

A metric space consists of a set of points and a distance function: ⟨ℙ, d: ℙ ⊛ ℙ ⟶ ℝ⟩

  • Distinct: ∀A,B∈ℙ d(A, B) = 0ℝ ⟺ A = B
  • Symmetric: ∀A,B∈ℙ d(A, B) = d(B, A)
  • Triangle: ∀A,B,C∈ℙ d(A, C) ≤ d(A, B) + d(B, C)
  • Non-negative: ∀A,B∈ℙ d(A, B) ≥ 0ℝ

Second That Motion

An isometry is a mapping (bijection) between one metric space and another that preserves distances.

∀A,B∈ℙ d(f(A), f(B)) = d(A, B)

A motion is just an isometry from some metric space to itself.

Mirror

All motions can be constructed using one or more mirrors.

v' = -u v u-1

This creates a mirror through the origin in the direction of non-zero vector u.

Double Mirror

When two mirrors intersect we rotate

This is a rotor: v' = (u w) v (u w)-1

Rotors

A product of two vectors is a rotor

v u = R = Rc + Rs U

  • R† = Rc + Rs U† = Rc - Rs U
  • |R|2 = R R† = Rc2 + Rs2 U U† + Rc Rs U† + Rs Rc U =
    Rc2 + Rs2 U U† + Rc Rs U - Rc Rs U     = Rc2 + Rs2

This is the equation for a circle in the plane decribed by bivector U

Unit Rotors

A unit rotor has a norm of one.

|R|2 = 12 = 1 = Rc2 + Rs2

Is this starting to look familiar?

cos(θ)2 + sin(θ)2 = 1

Euler's Formula

Consider: U2 = -1, U3 = -U, U4 = 1, U5 = U, ...

  • ex = 1 + x + x2/2 + x3/6 + ... + xn/n! + ...
  • eθU = 1 + θU + (θU)2/2 + (θU)3/6 + ...
  • eθU = 1 + θU - θ2/2 - θ3/6 U + ...
  • eθU = (1 - θ2/2 + ...) + (θ - θ3/6 + ...)U
  • eθU = cos(θ) + sin(θ)U

Euler's Rotor

All unit rotors can be expressed this way

  • I = eθ U = cos(θ) + sin(θ) U
  • I I† = (cos(θ) + sin(θ) U) (cos(θ) - sin(θ) U) = cos(θ)2 - cos(θ) sin(θ) U + sin(θ) cos(θ) U - sin(θ)2 U2 = cos(θ)2 + sin(θ)2 = 1

Euler's Rotor

R = |R| eθU = |R| (cos(θ) + sin(θ)U)

  • R-1 = R† / (R R†) = |R| (cos(θ) + sin(θ)U†) / |R|2 =
    (cos(θ) - sin(θ)U) / |R|     =
    (cos(-θ) + sin(-θ)U) / |R|     = e-θU / |R|
  • R v R-1 = (|R| eθU) v (e-θU / |R|) = eθU v e-θU

Rotor angles are all that matters in sandwich

Euclidian Rotor

Consider: v' = R v R-1

|v'|2 = v' v'† = (R v R-1) (R v R-1) = R v R-1 R v R-1 =
R v v R-1 = R v v† R-1 = |v|2 R R-1 = |v|2

Euclidian rotors preserve vector quadrance and therefore also length

Projective Geometry

Rotors always rotate around the origin. Projecting down from one extra dimension allows rotors to do this. This is a rotation around the point at infinity.

Minkowski Space

Until now all our basis vectors squared to one. When we introduce some basis vectors that square to negative one we get a mixed signature space.

x2 = y2 = z2 = 1 = -t2

t2 = -1

Anti-Commute Basis

Our axiom requires basis vectors to anti-commute.

  • (x y = -y x) ∧ (y z = -z y) ∧ (x z = -z x)
  • (x t = -t x) ∧ (y t = -t y) ∧ (z t = -t z)

Nothing about this changes in Minkowski space.

Spacetime

Convert time to distance using the speed of light (For example: c ⋍ 3×108 m/s)

v = vx x + vy y + vz z + vt c t

  • v v† = vx vx x x + vy vy y y + vz vz z z + vt c vt c t t + ...
  • v v† = vx vx x x + vy vy y y + vz vz z z + vt c vt c t t
  • v v† = vx2 + vy2 + vz2 - (vt c)2

This is the spacetime interval (mostly plus)

Spacetime Vectors

Spacetime vectors are characterized by their squared norm

  • Spacelike: v v† > 0 ⟹ vx2 + vy2 + vz2 > (c vt)2
  • Timelike: v v† < 0 ⟹ vx2 + vy2 + vz2 < (c vt)2
  • Lightlike: v v† = 0 ⟹ vx2 + vy2 + vz2 = (c vt)2

Lightlike vectors form the light cone.

Minkowski Rotors

A product of two Minkowsky vectors is a rotor (we'll assume vy = vz = 0 to keep things simple)

v u = (vx x + c vt t) (ux x + c ut t) =
vx ux x x + c vx ut x t + c vt ux t x + c2 vt ut t t =
vx ux - c2 vt ut + c (vx ut - vt ux) x t

v u = M = Mc + Ms x t

Minkowski Bivectors

Minkowski bivector are anti-commutative but do something interesting when squared:

  • (x t)2 = x t x t = -t x x t = -t t = -(-1) = 1
  • M M† = (Mc + Ms x t) (Mc - Ms x t) =
    Mc2 + Mc Ms x t - Mc Ms x t - Ms2 x t x t     = Mc2 - Ms2

Unit Minkowski Rotor

A unit Minkowski rotor has a norm of one

|M|2 = 12 = 1 = M M† = Mc2 - Ms2

This is the equation of a unit hyperbola

cosh(φ)2 - sinh(φ)2 = 1

Lorentz Rotor

Consider: (x t)2 = 1

  • eφ x t = 1 + φ x t + (φ x t)2/2 + (φ x t)3/6 + ...
  • eφ x t = 1 + φ x t + φ2/2 + φ3/6 x t + ...
  • eφ x t = (1 + φ2/2 + ...) + (φ + φ3/6 + ...) x t
  • eφ x t = cosh(φ) + sinh(φ) x t

Lorentz Rotor

Perhaps surprisingly, this is a unit rotor

  • L = eφ x t = cosh(φ) + sinh(φ) x t
  • L L† = (cosh(φ) + sinh(φ) x t) (cosh(φ) + sinh(φ) t x) = cosh(φ)2 + cosh(φ) sinh(φ) t x + sinh(φ) cosh(φ) x t + sinh(φ)2 x t t x = cosh(φ)2 - sinh(φ)2 = 1

Lorentz Boosts

M = |M| eφL = |M| (cosh(φ) + sinh(φ)L)

  • M-1 = M† / M M† = |M| (cosh(φ) + sinh(φ)L†) / |M|2 = (cosh(φ) - sinh(φ)L) / |M|
    = (cosh(-φ) + sinh(-φ)L) / |M|     = e-φL / |M|
  • M v M-1 = (|M| eφL) v (e-φL / |M|) = eφL v e-φL

    • Hyperbolic angles are all that matter in sandwich

Lorentz Boosts

Consider: v' = M v M-1

|v'|2 = v' v'† = (M v M-1) (M v M-1) = M v M-1 M v M-1 =
M v v M-1 = M v v† M-1 = |v|2 M M-1 = |v|2

Minkowski rotors preserve spacetime intervals
